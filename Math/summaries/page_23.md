## Page 23 — Adding Decimal Numbers & the Pairing Strategy (Numbers & Operations)

### Concepts
- Adding decimal numbers (kilometres and Rands/cents)
- Solving word problems involving addition of decimals
- Using a "pairing" strategy to add multiple decimal numbers efficiently (grouping numbers that make whole numbers or easy totals first)
- Working with South African currency (Rands and cents written as decimals, e.g., R6,50)
- Understanding distance in kilometres expressed as decimals

### Key Rules
- **Pairing strategy:** When adding several decimal numbers, look for pairs that combine to make a whole number or a "friendly" number first. For example, 1,2 + 1,8 = 3,0 (the decimals pair to make a whole). This makes the overall addition easier.
- **Adding decimals:** Line up the decimal commas and add as normal, carrying where necessary.
- **Money as decimals:** Rands and cents are written as decimals in South Africa (e.g., R6,50 means 6 Rand and 50 cents). Adding money follows the same rules as adding decimals.
- **Word problems:** Read carefully, identify the numbers to add, and compute the total.

### Questions & Answers

#### Question 1 — Triathlon Word Problem (Sihle)

*Sihle entered a triathlon. She had to swim 1,2 km, run 2,86 km and cycle 2,4 km. How many kilometres did she complete altogether?*

**Working:**
- 1,2 + 2,86 + 2,4
- Pairing: 1,2 + 2,4 = 3,6; then 3,6 + 2,86 = 6,46

> **Answer: 6,46 km**

#### Question 2 — Triathlon Word Problem (Busile)

*Busile extended the triathlon. He had to run 6,4 km, swim 3,2 km and cycle 2,4 km. How many kilometres did he complete altogether?*

**Working:**
- 6,4 + 3,2 + 2,4
- Pairing: 6,4 + 2,4 = 8,8 (the 0,4 + 0,4 pairs nicely but does not make a whole — alternatively: 3,2 + 2,4 = 5,6; then 6,4 + 5,6 = 12,0)
- Or simply: 6,4 + 3,2 = 9,6; then 9,6 + 2,4 = 12,0

> **Answer: 12,0 km (12 km)**

### Areas to Focus On
- **Identify pairs that make whole numbers:** The key skill on this page is spotting which decimal parts add to 1,0 (e.g., 0,2 + 0,8 = 1,0; 0,4 + 0,6 = 1,0; 0,5 + 0,5 = 1,0; 0,3 + 0,7 = 1,0; 0,1 + 0,9 = 1,0). Memorise these complement pairs.
- **Apply pairing to money:** The same strategy works with Rands and cents. Look for cent amounts that pair to make a whole Rand (e.g., R2,60 + R1,40 = R4,00).
- **Read word problems carefully:** Identify all the amounts that need to be added together before calculating. Do not miss any item.
- **Check decimal alignment:** When adding decimals, make sure the decimal commas are lined up so that tenths are added to tenths and units to units.
- **Practice mental addition with decimals:** The pairing strategy is a mental maths shortcut. Practice doing these additions in your head by pairing first, then adding the remaining number.
- **Verify your answer:** After using the pairing strategy, quickly estimate the total to check it makes sense (e.g., 1,2 + 2,4 + 0,8 is roughly 1 + 2 + 1 = 4, so 4,4 is reasonable).
