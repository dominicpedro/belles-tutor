## Page 17 — Addition Strategies: Breaking Up, Rounding & Compensating (Number & Pattern)

### Concepts
- Adding decimal numbers using the "breaking up" strategy (splitting numbers into parts to add mentally)
- Adding whole numbers using the "rounding and compensating" strategy (round one number to make addition easier, then adjust)
- Identifying pairs of numbers that add up to 100 (complementary pairs)
- Adding Rand amounts (South African currency) using rounding and compensating

### Key Rules
- **Breaking Up Strategy**: Split numbers into whole parts and decimal parts, add each separately, then combine.
  - Example: 1.2 + 3.4 = (1 + 3) + (0.2 + 0.4) = 4 + 0.6 = 4.6
- **Rounding and Compensating Strategy**: Round one number up (or down) to a convenient number, do the addition, then subtract (or add) the amount you rounded by.
  - Example: 39 + 85 -> round 39 up to 40, so 40 + 85 = 125, then subtract 1 -> 124
  - Example: 12.8 + 4.7 -> round 12.8 to 13, so 13 + 4.7 = 17.7, then subtract 0.2 -> 17.5
- **Pairs to 100**: Two numbers that add up to 100 are complementary pairs (e.g., 35 + 65 = 100).

### Areas to Focus On
- **Breaking up with decimals**: When adding decimals, be careful to line up the decimal parts correctly. A common mistake is forgetting to carry over (e.g., 1.8 + 2.9: the 0.8 + 0.9 = 1.7, so the answer is 4.7, not 3.7).
- **Rounding and compensating direction**: Always remember -- if you round UP, you must SUBTRACT the difference at the end. If you round DOWN, you must ADD.
- **Pairs to 100**: Practise these until they are automatic. A quick trick: the units digits must add to 10 (except 0+0), and the tens digits must add to 9 (or 10 if units add to 0). For example, 35 + 65: 5 + 5 = 10, carry 1, 3 + 6 + 1 = 10.
- **Rand amounts**: These questions use prices ending in .95, .98, .99, .48, .49 -- very common in shops. Round each price UP to the nearest whole Rand (or 50c), add, then subtract what you added. This is a real-life skill for estimating totals when shopping.
- **Three-number additions**: Questions like 5.2 + 6.3 + 1.5 or 9.3 + 2.1 + 4.5 require adding in steps. Look for pairs that make whole numbers first (e.g., 6.3 + 1.5 is not neat, but 5.2 + 1.5 = 6.7, then + 6.3 = 13.0).
- **Show your thinking**: For Questions 2 and 4, always write out the rounding step and the compensation step separately to earn full marks.
