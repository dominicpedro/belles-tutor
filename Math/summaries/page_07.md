## Page 7 — Soccer Ball Cost Tables & Saving Money (Number & Patterns)

### Concepts
- Ratio tables (building up multiplication patterns)
- Calculating cost from a rate table (unit rate and multiples)
- Proportion: using a cost-per-item table to find totals for larger quantities
- Fractions of an amount (fraction of pocket money)
- Saving and budgeting with fractions of money

### Key Rules
- **Ratio tables**: Each column multiplies the number of items by the cost per single item. If 1 soccer ball costs R56, then 2 cost R112, 3 cost R168, etc.
- **Reading a ratio table**: Find the closest values in the table and add them together to get costs for quantities not directly shown.
- **Fraction of an amount**: To find a fraction of an amount, divide the amount by the denominator and multiply by the numerator. For example, 1/10 of R20 = R20 / 10 = R2.
- **Cost = number of items x price per item**

### Areas to Focus On
- **Using ratio tables together**: Practice combining the "ones" table (1–9) with the "tens" table (10–90) to calculate costs for any number of items. Break the number into tens and ones.
- **Extending tables for hundreds**: For quantities like 400 or 438, extend the pattern by recognising that 400 = 10 x 40, then add remaining tens and ones from the tables.
- **Fractions of amounts**: Make sure to divide the whole amount by the denominator. For 1/10, always divide by 10. Double-check: 1/10 of R15 = R1.50 (not R0.75).
- **Careful arithmetic**: Several of the student's written answers appear to contain calculation errors. Practice breaking large multiplications into steps and checking each step.
- **Word-to-maths translation**: Question (f) connects the table work to pure multiplication (56 x 352), showing that ratio tables are really just organised multiplication. Recognising this link is important.
