## Page 25 — Number & Pattern (Patterns, Functions and Algebra)

### Concepts
- Recognising and extending geometric/visual patterns using square tiles
- Building a number pattern from a visual (picture frame) pattern
- Completing a table of values from a pattern
- Describing a calculation rule that links the frame number to the number of tiles
- Using a rule/formula to predict values for larger numbers (generalisation)

### Key Rules
- Each picture frame is a border of square tiles around a central picture
- Frame 1: the picture has 1 tile on each side touching it, plus 4 corner tiles => 4 x 1 + 4 = 8 tiles
- Frame 2: the picture has 2 tiles on each side touching it, plus 4 corner tiles => 4 x 2 + 4 = 12 tiles
- General rule: **Number of tiles = 4 x (frame number) + 4**
- This can also be seen as: each frame adds 4 more tiles than the previous frame (constant difference of 4)
- Pattern in the table: 8, 12, 16, 20, 24, ... (add 4 each time)

### Questions & Answers

**1.** Square tiles are used to make picture frames. The first 3 picture frames make a pattern.

**(a)** Draw the fourth and fifth picture frames in the pattern.

**Answer:**
- Frame 4: A border of tiles around the picture with 4 tiles on each side touching the picture, plus 4 corner tiles (total 20 tiles). The frame is larger than Frame 3.
- Frame 5: A border of tiles around the picture with 5 tiles on each side touching the picture, plus 4 corner tiles (total 24 tiles). The frame is larger than Frame 4.

**(b)** Complete the table.

| Picture frame number | 1 | 2  | 3  | 4  | 5  |
|----------------------|---|----|----|----|----|
| Number of tiles      | 8 | 12 | 16 | 20 | 24 |

**Answer:**
- Frame 1: **8** tiles (given)
- Frame 2: **12** tiles (given)
- Frame 3: **16** tiles (given)
- Frame 4: **20** tiles
- Frame 5: **24** tiles

**(c)** Dan is calculating the number of tiles needed for each picture frame.

Dan's explanation (speech bubble):
> "There are 4 sides. On each side of the frame, there are tiles touching the picture. For frame 1, one tile touches the picture on each side, so I say 4 x 1 plus 4 tiles on the 4 corners. That is 8. For frame 2, two tiles touch the picture on each side, so I say 4 x 2 plus 4 tiles on the 4 corners. That is 12."

**How does Dan calculate the number of tiles needed for picture frame 3?**

**Answer:** Dan says: 4 x 3 (three tiles touching the picture on each of the 4 sides) plus 4 (corner tiles) = 12 + 4 = **16 tiles**.

**(d)** How many tiles are needed for picture frame 30?

**Answer:** Using the rule: 4 x 30 + 4 = 120 + 4 = **124 tiles**.

### Areas to Focus On
- Practice drawing the next terms in a visual/geometric pattern by carefully observing what changes from one frame to the next
- When completing a table, first look for the constant difference between consecutive values (here it is +4 each time)
- Understand how to describe a pattern rule in words: identify what stays the same (the 4 corners) and what changes (the number of tiles on each side)
- Practise writing the general rule as a formula: Number of tiles = 4 x frame number + 4
- Apply the rule to calculate values for large frame numbers (like frame 30) without having to draw or count — this is the power of finding a rule
- Pay attention to how a visual pattern translates into a number pattern and then into a rule/formula
