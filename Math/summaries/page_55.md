## Page 55 — Mode and Dot Plots (Data Handling / Statistics)

### Concepts
- Understanding the **mode** of a data set (the value that occurs most often)
- Reading and interpreting **dot plots** (also called line plots)
- Collecting data and recording it on a dot plot
- Recognising that the mode does not necessarily describe the typical experience of every individual in a data set
- Understanding that a data set can have **no mode** if all values occur equally often

### Key Rules
- **Mode**: The value that appears **most frequently** in a data set. A data set can have more than one mode, or no mode at all.
- A **dot plot** uses dots stacked above a number line to show how often each value occurs. The value(s) with the tallest stack of dots is the mode.
- The mode does not mean that amount of time applies to everyone — it only identifies the most common value.
- A set of data has **no mode** when every value occurs the same number of times (i.e., all values have equal frequency).

### Areas to Focus On
- **Reading dot plots carefully**: Count each dot precisely; mistakes in counting lead to wrong answers for mode, totals, and comparisons.
- **Understanding what the mode tells us (and what it does not)**: The mode is the most frequent value, but it does not mean most people fall on that value. It is not the same as the mean or median.
- **"Or more" / "Or less" questions**: When a question says "90 minutes or more," include 90 and everything above it. When it says "60 minutes or less," include 60 and everything below it.
- **No-mode data sets**: Know that when all values appear with equal frequency, there is no mode. Be ready to construct an example of 10+ values with no mode.
- **Explaining reasoning in words**: Several questions require written explanations. Practice writing clear sentences that reference the data (e.g., "The mode is 90 but only 5 out of about 28 children watch 90 minutes, so it does not represent everyone").
