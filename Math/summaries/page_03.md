## Page 3 — Ratio Tables & Cost Calculations (Number & Pattern)

**Date on page:** 09-02-2026
**Source:** NumberSense Mathematics Workbook 17, Grade 5

---

### Concepts
- Ratio and proportion (mixing paint in a fixed ratio)
- Completing ratio tables by identifying the multiplicative relationship
- Reading and extending price/cost tables
- Using a rate table to calculate the cost of a given number of items
- Breaking a number into parts to find a total cost (decomposition strategy)

---

### Key Rules
- When two quantities are in a fixed ratio, you can find one from the other by multiplying or dividing by the same factor.
- In a ratio table, each column is a multiple of the base ratio.
- To find the cost of a larger number of packets, break the number into parts that appear in the table, find the cost of each part, then add.
  - Example: 7 packets = 2 + 5 packets, so cost = R12 + R30 = R42.
  - Example: 34 packets = 30 + 4 packets, so cost = R180 + R24 = R204.

### Areas to Focus On
- **Ratio and proportion:** Practice finding missing values when two quantities are in a fixed ratio. The key is to identify the multiplier between the given pair and apply it consistently.
- **Question 1 (b), (c), and (d) need correction:** The student's answers for these sub-questions are incorrect. Revisit the concept:
  - If blue : green = 4 : 3, then green = blue x (3/4) and blue = green x (4/3).
  - For 80 litres blue: green = 80 x 3/4 = 60 litres.
  - For 90 litres green: blue = 90 x 4/3 = 120 litres.
  - For 1 litre blue: green = 1 x 3/4 = 0.75 litres.
- **Decomposition strategy for cost tables:** Break larger numbers into parts that match values already in the table, then add the corresponding costs. This is a powerful mental maths strategy.
- **Double-check arithmetic:** Some written answers may have computational errors — always verify by checking with the ratio or by adding the table values.
- **Extend the table if needed:** For numbers not directly in the table, think of combinations (e.g., 8 = 5 + 3, or 8 = 4 + 4) and add their costs.
