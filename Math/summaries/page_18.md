## Page 18 — Fractions of Shapes & Growing Patterns (Number & Pattern)

### Concepts
- Identifying what fraction of a shape is shaded
- Understanding fractions as parts of a whole
- Recognising and extending a growing pattern made of squares
- Completing a table to record a number pattern
- Using a pattern rule to predict values for larger picture numbers

### Key Rules
- A fraction describes the number of shaded parts out of the total equal parts in a shape: **shaded parts / total equal parts**
- Shapes must be divided into **equal** parts for fractions to apply
- In a growing pattern, find the rule (how much is added each time) and use it to extend the pattern
- To predict a value far ahead in a pattern, express the rule as a formula (e.g., if the pattern adds 4 each time starting from 1: Number of squares = 1 + 4 x (picture number - 1), which simplifies to 4n - 3)

### Areas to Focus On
- **Simplifying fractions:** After finding the fraction shaded, always check if it can be simplified (e.g., 2/4 = 1/2, 4/8 = 1/2, 2/6 = 1/3)
- **Counting equal parts carefully:** Make sure to count ALL parts of the shape (both shaded and unshaded) to get the correct denominator -- the student may have miscounted on question 1e
- **Identifying the pattern rule:** In growing patterns, calculate the difference between consecutive terms (here: 5-1=4, 9-5=4, so +4 each time)
- **Writing a formula for the pattern:** Practice turning "add 4 each time, starting at 1" into the formula 4n - 3, so you can find any term without drawing all the pictures
- **Applying the formula to large numbers:** For part (c), you cannot draw 24 pictures -- you must use the rule. Practise substituting into the formula: 4 x 24 - 3 = 93
- **Check your arithmetic:** The student's working shows "98 x 93" on the side -- make sure the multiplication or formula substitution is done carefully and clearly
