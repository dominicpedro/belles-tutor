## Page 38 — Transformations: Rotation, Reflection & Translation (Space & Shape)

### Concepts
- Understanding three types of geometric transformations: **rotation**, **reflection**, and **translation**
- Creating patterns by applying transformations to an arrow/flag shape
- Recognising the difference between rotating (turning), reflecting (flipping), and translating (sliding) a shape
- Extending patterns that use each type of transformation
- Using a cut-out shape on a dotted grid to physically perform transformations

### Key Rules
- **Rotation (turning):** The shape is turned around a point. The orientation changes (it can face up, down, left, right, or at angles) but the shape itself stays the same size and form. The blue (top) row demonstrates this -- the arrow shape is turned to different positions (facing right, left, up, down, and diagonal orientations).
- **Reflection (flipping):** The shape is flipped over a line (mirror line). It produces a mirror image. The yellow (middle) row demonstrates this -- the arrow shape alternates between its original orientation and its mirror image (flipped horizontally), creating a repeating reflected pattern.
- **Translation (sliding):** The shape is slid (moved) in one direction without turning or flipping. The shape keeps exactly the same orientation. The red (bottom) row demonstrates this -- the arrow shape is simply slid along, always facing the same direction, maintaining the same orientation throughout.

### Questions & Answers

**Question 1:** Trace this shape onto a piece of card and cut it out.

**Answer:** This is a hands-on instruction. The student must trace the arrow/flag shape (shown at the top-left of the page) onto card and cut it out to use as a template.

**Question 2:** Use the shape that you cut out to extend each of the patterns.

**Answer:**
- **Row 1 (Blue -- Rotation):** Continue the pattern by rotating the arrow shape. The student has completed this by drawing blue arrow shapes in various rotated orientations (turned at different angles) continuing the sequence across the grid.
- **Row 2 (Yellow -- Reflection):** Continue the pattern by reflecting the arrow shape. The student has completed this by drawing yellow arrow shapes that alternate between the original and its mirror image (flipped horizontally).
- **Row 3 (Red -- Translation):** Continue the pattern by translating (sliding) the arrow shape. The student has completed this by drawing red arrow shapes that all face the same direction, simply moved/slid along the row.

---

**Question 3:** Explain what you did with the shape to make each of the patterns. How was what you did with the shape the same and different for each of the patterns?

**Answer (expected response):**
- For the **Rotation** pattern, I turned the shape around to different positions each time. The shape faced different directions (left, right, up, down).
- For the **Reflection** pattern, I flipped the shape over each time. The shape alternated between facing one way and facing the opposite way, like a mirror image.
- For the **Translation** pattern, I slid the shape along without turning or flipping it. Every shape faces the same direction.
- **Same:** In all three patterns, I used the exact same shape and it stayed the same size. The shape was never stretched or changed.
- **Different:** In rotation the shape was turned; in reflection the shape was flipped to create a mirror image; in translation the shape was only slid along and never turned or flipped.

### Areas to Focus On
- Be able to clearly **name and distinguish** the three transformations: rotation = turn, reflection = flip, translation = slide
- Practice identifying which transformation was used by looking at how a shape changes (or does not change) its orientation in a pattern
- Remember that in a **translation**, the shape never changes direction -- it always looks exactly the same, just in a different position
- Remember that in a **reflection**, the shape looks like a mirror image -- if the original points right, the reflection points left
- Remember that in a **rotation**, the shape is turned around a point -- it can end up facing any direction
- Use physical cut-outs or tracings to practise each transformation hands-on, as the page instructs
- When extending patterns, first identify which transformation is being used, then apply that same transformation consistently to continue the sequence
- Be ready to **explain in words** what each transformation does -- this page specifically asks for written explanation (Question 3)
