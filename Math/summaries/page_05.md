## Page 5 — Matchstick Shape Patterns (Number & Pattern)

### Concepts
- Building geometric shapes (triangles, squares, pentagons, hexagons) from matchsticks
- Identifying and extending numeric patterns in a table
- Using flow diagrams to represent number patterns as rules (input -> operation -> output)
- Finding the relationship between the picture number and the number of matches needed
- Applying a pattern rule to predict values for larger numbers

### Key Rules
- **Triangle pattern**: Each triangle uses 3 matches. For picture number n: matches = n x 3 (i.e., 3, 6, 9, 12, 15, ...)
- **Square pattern**: Each square uses 4 matches. For picture number n: matches = n x 4 (i.e., 4, 8, 12, 16, 20, ...)
- **Pentagon pattern**: Each pentagon uses 5 matches. For picture number n: matches = n x 5 (i.e., 5, 10, 15, 20, 25, ...)
- **Hexagon pattern**: Each hexagon uses 6 matches. For picture number n: matches = n x 6 (i.e., 6, 12, 18, 24, 30, ...)
- **Flow diagram format**: Input (picture number) -> x (number of sides) -> Output (number of matches)
- To find the number of matches for any picture number, multiply the picture number by the number of sides of the shape.

### How the page is laid out
The page shows pictures of shapes made from matchsticks growing in a pattern (1 shape, then 2, then 3, etc.). Students must complete a table and flow diagrams.

**Question b:** Dan is calculating the number of matches needed for 16 triangles.

Dan says: "I know that 16 = 2 x 8. From my table I know that I need 17 matches for 8 triangles, so for 16 triangles I need 2 x 17 which is 34 matches."

**Is Dan correct? Explain.**

**Answer:** Dan is **not correct**. Dan's error is that he used the wrong value from the table and applied faulty reasoning. The correct method is:
- The rule for triangles is: number of matches = picture number x 3
- For 16 triangles: 16 x 3 = **48 matches**
- Dan's mistake was thinking the pattern works by doubling (i.e., that if you double the number of shapes, you double the matches from a smaller number). While 2 x 24 = 48 would actually work for separate triangles, Dan used 17 (which is not the correct value for 8 triangles from the table — 8 x 3 = 24, not 17). Dan appears to have confused this with a different pattern rule (likely the n x 3 + 1 rule from a previous pattern on an earlier page).

The correct number of matches for 16 triangles is **48**.

### Areas to Focus On
- **Understand the pattern rule**: For separate (non-joined) shapes, the number of matches = number of shapes x number of sides of the shape
- **Flow diagrams**: Practice reading and completing flow diagrams — identify the operation (multiply by the number of sides) that converts input to output
- **Extending patterns**: Be confident predicting values beyond the table by applying the rule, not just reading from the table
- **Checking reasoning**: When someone claims a shortcut (like doubling), verify it against the actual rule. Doubling the input should double the output in a multiplicative pattern, but only if you use the correct values
- **Common mistakes**: Do not confuse pattern rules from different questions (e.g., mixing up "n x 3" with "n x 3 + 1" from joined-shape patterns on other pages)
- **Tables and flow diagrams**: Practice filling in both representations — they show the same pattern in different formats
