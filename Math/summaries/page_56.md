## Page 56 — Mode from Dot Plots (Data Handling)

### Concepts
- Reading dot plots (line plots) to identify data values and their frequencies
- Determining the mode (most frequent value) from a dot plot
- Understanding that adding data points to a dot plot can change the mode
- Recognising that the mode is the value with the tallest column of dots

### Key Rules
- The **mode** is the value that appears most often in a data set
- On a dot plot, the mode is the value on the number line that has the most dots stacked above it
- A data set can have more than one mode if two or more values share the highest frequency
- Adding new data points to a dot plot can change which value is the mode

### Areas to Focus On
- **Finding the mode visually:** Scan across a dot plot and look for the tallest stack of dots — the value beneath it is the mode
- **Counting dots carefully:** Miscounting by even one dot can lead to identifying the wrong mode; count each column methodically
- **Multimodal data:** Be aware that if two or more values share the same highest frequency, the data set has more than one mode — list all of them
- **Effect of adding data:** When the question asks you to add dots to change the mode, think strategically — add all 5 dots to a single value (or concentrate them) to make that value the new mode
- **Reading scales:** Pay close attention to the number line — dot plots can use scales of 1, 5, 10, or other intervals; read the axis labels carefully
- **Data Handling terminology:** Know the difference between mode (most frequent), median (middle value), mean (average), and range (difference between highest and lowest)
