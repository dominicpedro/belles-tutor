# PM = Plus

**Use when:** Converting between 12-hour and 24-hour time.

**Trigger:** See "pm"? PLUS 12. See a big number? MINUS 12.

---

## To 24-hour time

| Time | Rule | Example |
|------|------|---------|
| Morning (am) | Keep it, write 2 digits | 7:20 am = **07:20** |
| Afternoon/night (pm) | **Plus 12** | 3:45 pm = **15:45** |
| Midnight | = **00:00** | |
| Noon | = **12:00** | |

## Back to 12-hour time

| 24-hour number | Rule | Example |
|----------------|------|---------|
| Less than 12 | It's **am** | 07:20 = 7:20 **am** |
| 12 or more | **Minus 12**, it's **pm** | 15:45 - 12 = 3:45 **pm** |

---

## Don't forget

ALWAYS write **am** or **pm** when going back to 12-hour time. No label = lost mark.
