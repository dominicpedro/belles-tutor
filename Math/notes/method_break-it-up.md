# Break It Up

**Use when:** Adding big numbers (whole numbers or decimals). Also works for cost tables.

**Trigger:** "Break it into friendly pieces, add them up."

---

## Steps

1. BREAK each number into hundreds, tens, and ones (or rands and cents)
2. ADD each part separately (hundreds + hundreds, tens + tens, ones + ones)
3. COMBINE the totals

---

## On Paper (test)

```text
368 + 245 = ?

300 + 200 = 500
 60 +  40 = 100
  8 +   5 =  13
             ---
             613
```

## Typed (study sessions)

```text
368+245: 300+200=500, 60+40=100, 8+5=13 => 613
```

---

## With decimals

Same idea -- break at the comma:

```text
3,7 + 2,8: 3+2=5, 0.7+0.8=1.5 => 6,5
```

---

## With cost tables

Need 28 items at R15 each?

```text
20 x R15 = R300
 8 x R15 = R120
           ----
           R420
```
